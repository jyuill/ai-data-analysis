# Reviewer package
Package produced by Posit team member to use AI to review R code and provide feedback.
https://github.com/simonpcouch/reviewer

-- NOTE: a very nascent experimental project, not even beta --

## Install

Install from github:
- As of Mar 2026: `install.packages('simonpcouch/reviewer')`

## Usage

`reviewer::review("path to your R script")`

See: `reviewer.R`
